Toggling themes based on screen size

If you find yourself battling with convoluted media queries and complex styles to switch between mobile and desktop, possibly because the styles in mobile happen to be very different to those of large screens, then you might find this approach useful.

In the following way, we can toggle the desktop/mobile themes whenever appropriate.

<link id="mobile-theme" rel="stylesheet" media="screen and (max-width: 700px)" href="themes/mobile/mobile.bundled.css" />
<link id="desktop-theme" rel="stylesheet" media="screen and (min-width: 701px)" href="themes/desktop/desktop.bundled.css" />

This is a simple way to toggle the theme based on the screen size. You can see the difference by resizing the browser window.

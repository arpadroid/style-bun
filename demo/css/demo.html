Installation

npm install @arpadroid/style-bun

Basic Usage

import { ThemesBundler } from '@arpadroid/style-bun';

const bundler = new ThemesBundler({
  themes: [
    { path: './src/themes/default' },
    { path: './src/themes/dark' }
  ],
  minify: process.env.NODE_ENV === 'production'
});

await bundler.bundle();

Simply toggle the disabled attribute to switch themes:

Use CSS media queries to automatically load different themes based on screen size:

                    <link id="mobile-theme" rel="stylesheet" media="screen and (max-width: 700px)" href="themes/mobile/mobile.bundled.css" />
                    <link id="desktop-theme" rel="stylesheet" media="screen and (min-width: 701px)" href="themes/desktop/desktop.bundled.css" />

This approach automatically toggles themes based on viewport size - resize your browser window to see the difference!
